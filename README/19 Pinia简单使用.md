## 19 Pinia简单使用

14. Pinia

	14.1 Vuex和pinia的区别

		参考网址： https://github.com/vuejs/rfcs/pull/271 

		1. pinia没有mutations，只有：state、getters、actions
		2. pinia分模块不需要modules（之前vuex分模块需要modules）
		3. pinia体积更小（性能更好）
		4. pinia可以直接修改state数据

	14.2 pinia使用

		官方网址：https://pinia.vuejs.org/
