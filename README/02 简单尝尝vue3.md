## 三、vue2 和 vue3 区别 
### 3.1 v-if 与 v-for 的优先级对比 

2.x 版本中 v-for > v-if 

3.x 版本中 v-if > v-for 

### 3.2 v-for 中的 Ref 数组 
vue2.x 会自动把ref填充内容 
vue3.x 需要手动添加
{{ item }}
methods:{ setItemRef(el){ this.arr.push( el ); } } 

### 3.3 $children 
vue2.x : 访问当前实例的子组件 
vue3.x : 在 3.x 中，$children 已被移除，且不再支持。 
设置： 访问：this.$refs.hw
